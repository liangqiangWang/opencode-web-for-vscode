OpenCode 帮助文档

什么是 OpenCode？

OpenCode 是一个强大的 AI 编程助手，可以帮助你：

  • 快速编写和理解代码
  • 自动生成代码片段
  • 提供智能代码补全
  • 解答编程问题

安装指南

系统要求

  • Node.js >= 16.0.0
  • npm 或 yarn
  • VSCode >= 1.70.0

安装步骤

使用 npm 全局安装：

npm install -g opencode-ai

或使用 yarn：

yarn global add opencode-ai

验证安装

安装完成后, 重启一下 vscode, 运行以下命令验证：

opencode --version

使用指南

启动 OpenCode

  1. 打开 VSCode 侧边栏的 OpenCode-Web 面板
  2. 点击"启动 OpenCode"按钮
  3. 等待服务启动完成
  4. 开始使用

发送代码到 OpenCode 终端

  1. 在编辑器中选中代码
  2. 右键点击选中的代码
  3. 选择"添加到 OpenCode 终端"
  4. 代码将自动发送到 OpenCode 终端
提示： 确保 OpenCode 服务正在运行，否则无法发送代码。

常见问题

Q: OpenCode 无法启动？

A: 请检查端口 4099 是否被占用，或者尝试修改配置中的端口号。

Q: 连接超时怎么办？

A: 可尝试点击顶部栏的"重启进程"按钮，会杀掉opencode进程，重新启动

Q: 已安装 OpenCode，但仍提示"OpenCode 未安装"？

A: 这可能是安装状态缓存导致的检测失败。请尝试：

  1. 点击侧边栏顶部的刷新连接按钮
  2. 如果无效，重启 VSCode（Ctrl+Shift+P → "Reload Window"）
  3. 验证安装：在终端运行 opencode --version

Q: OpenCode Web 界面的颜色与 VSCode 主题不匹配？

A: OpenCode Web 目前无法自动适配 VSCode 主题。需要手动调整：

  1. 点击 OpenCode 界面右上角的设置图标
  2. 在外观设置中切换颜色模式（亮色/暗色/自动）
  3. 选择与 VSCode 当前主题匹配的模式

Q: OpenCode Web 模式下无法复制粘贴内容？

A: 由于 Webview 以 iframe 方式嵌入 VSCode，浏览器跨域安全限制导致键盘快捷键（Ctrl+C/Ctrl+V）无法正常工作。

解决方案：

  • 使用 OpenCode 界面内的复制按钮进行复制
  • 或在独立终端中使用 OpenCode TUI（点击侧边栏"打开 TUI 终端"）
