OpenCode Help Documentation

What is OpenCode?

OpenCode is a powerful AI coding assistant that can help you:

  • Write and understand code quickly
  • Automatically generate code snippets
  • Provide intelligent code completion
  • Answer programming questions

Installation Guide

System Requirements

  • Node.js >= 16.0.0
  • npm or yarn
  • VSCode >= 1.70.0

Installation Steps

Install globally using npm:

npm install -g opencode-ai

Or using yarn:

yarn global add opencode-ai

Verify Installation

After installation, restart VSCode and run the following command to verify:

opencode --version

Usage Guide

Launch OpenCode

  1. Open the OpenCode-Web panel in the VSCode sidebar
  2. Click the "Launch OpenCode" button
  3. Wait for the service to start
  4. Start using it

Send Code to OpenCode Terminal

  1. Select code in the editor
  2. Right-click on the selected code
  3. Select "Append to OpenCode Terminal"
  4. The code will be automatically sent to the OpenCode terminal
Tip: Make sure the OpenCode service is running, otherwise you cannot send code.

FAQ

Q: OpenCode cannot start?

A: Please check if port 4099 is occupied, or try modifying the port number in the configuration.

Q: What to do if connection times out?

A: Try clicking the "Restart Process" button in the top bar, which will kill the opencode process and restart it.

Q: OpenCode is installed, but still shows "OpenCode not installed"?

A: This may be due to installation status caching. Please try:

  1. Click the Refresh Connection button at the top of the sidebar
  2. If that doesn't work, restart VSCode (Ctrl+Shift+P → "Reload Window")
  3. Verify installation: run opencode --version in terminal

Q: OpenCode Web interface colors don't match VSCode theme?

A: OpenCode Web currently cannot automatically adapt to VSCode themes. You need to manually adjust:

  1. Click the Settings icon in the upper-right corner of OpenCode interface
  2. Switch Color Mode (Light/Dark/Auto) in appearance settings
  3. Select a mode that matches your current VSCode theme

Q: Cannot copy/paste content in OpenCode Web mode?

A: Due to browser cross-origin security restrictions (iframe embedding), keyboard shortcuts (Ctrl+C/Ctrl+V) may not work properly.

Solutions:

  • Use the Copy button inside the OpenCode interface
  • Or use OpenCode TUI in a standalone terminal (click "Open TUI Terminal" in the sidebar)
